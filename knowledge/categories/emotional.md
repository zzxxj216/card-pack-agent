# Category Playbook: Emotional

**L1**: `emotional`
**Status**: v0.2 — Minimal scaffold. Unverified mechanism recipes,
default-mechanism preferences, visual notes, and open-question speculation
removed. Mechanism selection is now topic-driven.
**Validation progress**: 0 / 30 packs

---

## 1. Scope

### 1.1 In scope
- Packs anchored on an **emotion / psychological state** as the core,
  independent of a specific external trigger
- Common emotional anchors:
  - Loneliness (especially big-city / post-college / late-20s)
  - Burnout (work, caregiving, parenting)
  - Imposter / comparison exhaustion (social media fatigue)
  - Grief at low intensity (ambient loss, missing a past self)
  - Quiet joy / gratitude / contentment
  - Anxiety / overthinking loops
  - Healing / recovery / post-therapy realizations
  - Nostalgia for past selves / past eras

### 1.2 Out of scope
- An emotion triggered by a **specific holiday** → `festival`
- An emotion triggered by a **specific news event** → `trending_event`
- "How to overcome X emotion" step-by-step → `knowledge`
- Personal transformation stories → `growth`
- Relational emotion (between specific people) → `relationship`

### 1.3 Intensity range
Stay below the saturation line (see `global_style_guide.md §5`). Emotional
packs easily tip into over-emotion, which both the Evaluator flags and
audiences reject as performative.

---

## 2. Emotional-specific taboos

1. **No diagnosis language** ("this is depression", "you have anxiety") —
   we describe, we don't clinically label
2. **No forced uplift**: don't end every pack with "you got this"
3. **Over-emotion ceiling** — hard rule: 3+ consecutive cards with
   "cry / tears / sobbing / broken / wrecked / devastated" flags the pack
4. **No pathologizing normal feelings**: being tired is not burnout; being
   sad is not depression
5. **No "top 10 signs you have X"** framing — that's `knowledge`, not
   `emotional`
6. **Avoid saturated openers**:
   - "We've all been there..."
   - "Every [20-something / woman / creator] knows this feeling..."
   - "POV: you're literally me"
